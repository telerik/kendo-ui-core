# Binding

The following example demonstrates how to create a CheckBoxGroup and bind it to an array of strings.

```dojo
   <ul id="checkboxgroup"></ul>

    <script>
        $("#checkboxgroup").kendoCheckBoxGroup({
            items: [ "one", "two", "three" ]
        });
    </script>
```
