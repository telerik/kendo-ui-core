# Binding

The following example demonstrates how to create a RadioGroup and bind it to an array of strings.

```dojo
   <ul id="radiogroup"></ul>

    <script>
        $("#radiogroup").kendoRadioGroup({
            items: [ "one", "two", "three" ]
        });
    </script>
```
