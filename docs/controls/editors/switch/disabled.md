# Disabled Switch

By default, the Switch is enabled.

The following example demonstrates how to disable the Switch.

    <input type="checkbox" id="switch" />

    <script>
        var switchInstance = $("#switch").kendoSwitch({
                enabled: false
            });
    </script>
