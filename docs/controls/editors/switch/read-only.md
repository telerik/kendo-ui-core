# Read-Only Switch

By default, the Switch is in an active state.

The following example demonstrates how to render the Switch in its read-only state.

    <input type="checkbox" id="switch" />

    <script>
        var switchInstance = $("#switch").kendoSwitch({
                readonly: true
            });
    </script>
