bounceVelocityThreshold: Number(default: 2.5)
The velocity threshold after which a swipe will result in a bounce effect.
duration: Number(default: 300)
The milliseconds that take the ScrollView to snap to the current page after released.
velocityThreshold: Number(default: 1)
The velocity threshold after which a swipe will navigate to the next page (as opposed to snapping back to the current view). The swipe velocity equal the pixels per millisecond change for a swipe.
