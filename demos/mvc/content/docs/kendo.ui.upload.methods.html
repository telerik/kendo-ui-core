• disable ()
    Disables the upload.

    Example

    var upload = $("#upload").data("kendoUpload");

// disables the upload
upload.enable();
  • enable ()
    Enables the upload.

    Example

    var upload = $("#upload").data("kendoUpload");

// enables the upload
upload.enable();
  • toggle (enable)
    Toggles the upload enabled state.

    Example

    var upload = $("#upload").data("kendoUpload");

// toggles the upload enabled state
upload.toggle();

    Parameters

    enable: Boolean
    (Optional) The new enabled state.
