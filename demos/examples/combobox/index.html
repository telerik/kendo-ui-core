Basic usage

Creating a ComboBox from <input />

Source:

<input id="input" value="1" />

<script>
    $("#input").kendoComboBox([
        { text: "Item 1", value: "1" },
        { text: "Item 2", value: "2" },
        { text: "Item 3", value: "3" }
    ]);
</script>

Creating a ComboBox from <select />

Source:

<select id="select">
    <option>Item 1</option>
    <option>Item 2</option>
    <option>Item 3</option>
</select>

<script>
    $("#select").kendoComboBox();
</script>
