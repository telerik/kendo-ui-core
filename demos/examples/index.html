Aeroviewr

This rich HTML5 integration demo shows Kendo UI in action to interact with Flickr photo streams. The demo features several Kendo UI controls, Kendo UI DataSource, and other Kendo UI framework elements. Check it out online and in mobile browsers to see how Kendo UI helps create universally accessible apps.

Mobile Device Compatibility

Kendo UI is built to work seamlessly across desktop browsers and mobile touchscreen devices. Kendo UI offers full support for touch interactions and is tested against the following mobile platform browsers:

  • Android 2.x+
  • iOS 3.x+
  • BlackBerry OS 6.x+
  • webOS X.x+

Supported browsers

  • Internet Explorer 6+
  • Firefox 3+
  • Safari 4+
  • Chrome
  • Opera 10+
