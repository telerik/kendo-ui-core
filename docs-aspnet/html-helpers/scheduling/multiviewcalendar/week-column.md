# Week Number Column

In the MultiViewCalendar, you can render a column which displays the number of the weeks within the current `month` view.

```Razor
      @(Html.Kendo().MultiViewCalendar()
          .Name("MultiViewCalendar")
          .WeekNumber(true)
      )
```
