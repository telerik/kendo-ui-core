# Keyboard Navigation

The Telerik UI for {{ site.framework }} PromptBox component provides comprehensive keyboard navigation support, allowing users to interact efficiently without relying on a mouse.

Keyboard navigation enhances accessibility and improves productivity for power users who prefer keyboard-based workflows.

## Supported Keyboard Shortcuts

The PromptBox supports the following keyboard shortcuts by default:

| Shortcut | Action |
|----------|--------|
| `Enter` | Submit the prompt (configurable) |
| `Ctrl+Enter` / `Cmd+Enter` | Submit the prompt in multi-line mode |
| `Shift+Enter` | Insert a new line in multi-line mode |
| `Escape` | Clear the input or cancel streaming |
| `Tab` | Move focus to the next focusable element |
| `Shift+Tab` | Move focus to the previous focusable element |
| `Ctrl+A` / `Cmd+A` | Select all text |
| `Ctrl+C` / `Cmd+C` | Copy selected text |
| `Ctrl+V` / `Cmd+V` | Paste text from clipboard |
| `Ctrl+X` / `Cmd+X` | Cut selected text |
| `Ctrl+Z` / `Cmd+Z` | Undo last action |
| `Ctrl+Y` / `Cmd+Shift+Z` | Redo last action |
