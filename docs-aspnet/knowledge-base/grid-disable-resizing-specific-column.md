## Description

How can I disable column resizing for specific columns in the {{ site.product }} Grid?

## Solution 

By default, column resizing is enabled or disabled for all Grid columns.

For a runnable example on how to prevent resizing for the **Freight** column, refer to this [Telerik REPL Project.](https://netcorerepl.telerik.com/QmuvvBFe590oDUnP21)
